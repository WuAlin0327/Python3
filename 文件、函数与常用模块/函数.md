# 函数
* 函数是指将一组语句的集合通过一个名字（函数名）封装起来，要想执行这一个函数，只需调用其函数名即可
* 语法
```
def sayhi(): # 函数名
    print("Hello world")

sayhi() # 调用函数
```
* 特性
1. 减少重复代码
2. 使程序变的可扩展
3. 使程序变的易维护

##### 形参与实参
```
def sum(x,y): # 这个x,y为形参
    res = x*y
sum(2,5)# 这个2,5为实参
```
